Introducing Tools We May Use! Great tools used in industrial areas

During the course of developing projects, we would introduce and utilize several different IDEs and other tools.

GitHub

GitHub is a online code hosting service utilizing Git as version control system. We manage, publish and distribute all of resources and codes through GitHub.

  • Easy project management
  • Easy contribution tracking
  • Easy version control

iOS Development

Xcode

Xcode is a powerful and essential integrated development environment (IDE) for developing native iOS, MacOS, WatchOS applications. We will develop some projects using Xcode and the modern programming language - Swift.

Swift

Android Development

Android Studio

Android Studio is the official integrated development environment (IDE) for Google's Android operating system, built on JetBrains' IntelliJ IDEA software and designed specifically for Android development. We will use Kotlin instead of Java to build our apps for Android.

Kotlin XML

Windows Development

Visual Studio

Visual Studio is a powerful and intelligent IDE for developing native Windows apps and way beyond that. Lots of platforms exist for native Windows apps, such as Windows Forms, WPF and UWP. We will develop apps for UWP, the newest platform, utilizing Visual Studio, C# and XAML.

C# XAML

Want More?

Native development can be fancy, but it will be hard to develop and manage several different native apps for different platforms! With modern technology, we may also introduce to you another powerful framework!

"Flutter"

Advanced Building

Flutter is equipped with advanced building technology. Hot reload can help you easily and quickly see the change just made.

Cross the Platform

You can easily build up a app for diverse platforms, both iOS and Android, with only one version of code. Managable and flexible.

Optimized Performance

Flutter compiles one version of code into several native versions for targeted OS. Less code but better performance.
